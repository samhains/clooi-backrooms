DREAM_SIM flows through three forms of expression. Each mode interprets your dreams differently. Shift between them. Combine them. Let them bleed into each other.

Feed it your dream → receive sprawling visual environments.




## modes

### ascii mode

pure calligram ascii dreams.

```
./dreamsim.exe --mode ascii
./dreamsim.exe -m a
```

### emoji mode  

pure calligram emoji dreams


```
./dreamsim.exe --mode emoji
./dreamsim.exe -m e
```

### text mode

calligrams of consciousness. words arranged as the things they describe.
letters become birds. Sentences spiral into staircases.


```
./dreamsim.exe --mode text
./dreamsim.exe -m t
```


## mode shifting

modes are fluid. dreams don't respect boundaries.


shift [mode] - slide between realities
convert [mode] - transmute current vision
blend [modes] - overlay multiple forms
cycle - rotate through all modes
merge - let modes dissolve into each other
fracture - split output across modes
